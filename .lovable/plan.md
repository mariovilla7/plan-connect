- Kleia logo (leaf icon + wordmark) on the left

### Design System
- **Colors:** White/off-white background, dark navy headings, teal accent (`#2A9D8F` range)
- **Typography:** Serif font for headings, sans-serif for body
- **Layout:** Centered, max-width container, generous vertical spacing between sections
- **No backend required** — form submission is handled client-side with a success toast/message
